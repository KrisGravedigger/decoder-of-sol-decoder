Win Rate vs Required Win Rate Analysis

This chart visualizes the Expected Value (EV) framework, comparing the historical performance of the strategy against the theoretical minimum win rate required for long-term profitability.

  • The Required Win Rate (dashed line) is the theoretical percentage of winning trades needed to break even, calculated as: SL / (TP + SL).
  • The Historical Win Rate (solid line) is the actual observed win rate from the simulation data (calculated based on positive PnL).
  • For a Stop Loss level to be viable, the Historical Win Rate must be above the corresponding Required Win Rate line. Viable zones (if found) are shaded green.

Dynamic Stop Loss Floor Analysis

This table identifies the 'Deepest Viable SL' for each Take Profit level, ensuring the strategy maintains a positive Expected Value (EV). This analysis helps set a rational safety floor.

  • Deepest Viable SL: The largest loss percentage that the historical win rate can sustain while remaining profitable (Historical WR > Required WR).
  • Safety Margin: The difference between Historical and Required win rates. This is color-coded to indicate statistical confidence: Green (≥10%), Yellow (5-10%), Red (<5%).
